La cláusula 6.2 establece que la organización debe definir objetivos de seguridad de la información en las funciones y niveles pertinentes.

- Ser coherentes con la política de seguridad.

::title::
Criterio SMART aplicado al SGSI

::content::
Un objetivo de seguridad efectivo debe ser:

- S — Specific (Específico): aborda una situación concreta del SGSI.
- M — Measurable (Medible): incluye indicadores cuantificables.
- A — Achievable (Alcanzable): es realista con los recursos disponibles.
- R — Relevant (Relevante): alineado con la estrategia organizacional.
- T — Time-bound (Con plazo): tiene fecha de cumplimiento clara.

Ejemplo no SMART: "Mejorar la seguridad de la información".

Ejemplo SMART: "Reducir en un 30% los incidentes de phishing reportados antes del 31 de diciembre de 2026, mediante capacitación obligatoria al 100% del personal administrativo".

::title::
Pasos clave para establecer objetivos efectivos

::left::
Definición y alineación

1. Analizar el contexto organizacional.
2. Identificar partes interesadas y sus expectativas.
3. Alinear los objetivos con la estrategia y la política de seguridad.
4. Considerar requisitos legales y regulatorios.
5. Documentar los objetivos.

::right::
Operación y seguimiento

6. Asignar responsables y recursos.
7. Definir métricas e indicadores.
8. Establecer plazos concretos.
9. Comunicar los objetivos a todos los niveles.
10. Revisar, ajustar y mejorar continuamente.

::title::
Riesgos, coordinación y alta dirección

::content::
Análisis de riesgos: la planificación debe incluir la identificación y evaluación de riesgos que puedan obstaculizar el logro de los objetivos. Esto permite priorizar y diseñar controles efectivos.

Coordinación interna: los objetivos deben integrarse con la gestión de riesgos, el cumplimiento normativo y la administración de recursos humanos. La gestión aislada compromete la eficacia.

Alta dirección: debe garantizar la asignación de recursos, comunicar los objetivos a todos los niveles y fomentar una cultura de seguridad. No basta con aprobar: debe participar activamente.

::title::
Código Python — Parte 2

::content::
```python
# Función que valida si un objetivo cumple el criterio SMART completo
def es_smart(obj):
    """Retorna True si el objetivo cumple los cinco criterios SMART."""
    return (
        obj["especifico"] and
        obj["medible"] and
        obj["alcanzable"] and
        obj["relevante"] and
        obj["plazo"] is not None
    )

# Función que calcula días restantes hasta el plazo
def dias_restantes(plazo):
    """Calcula días desde hoy hasta el plazo del objetivo."""
    if plazo is None:
        return None
    hoy = datetime.now().date()
    fecha_plazo = datetime.strptime(plazo, "%Y-%m-%d").date()
    return (fecha_plazo - hoy).days

# Generación del reporte de cumplimiento del SGSI
print("=" * 60)
print("REPORTE DE OBJETIVOS DEL SGSI - ISO/IEC 27001")
print("=" * 60)

smart_count = 0
avance_total = 0

for obj in objetivos_sgsi:
    estado_smart = "SMART" if es_smart(obj) else "NO SMART"
    dias = dias_restantes(obj["plazo"])
    info_plazo = f"{dias} días restantes" if dias is not None else "Sin plazo"

    print(f"\n[{obj['id']}] {obj['descripcion']}")
    print(f"   Responsable : {obj['responsable']}")
    print(f"   Estado SMART: {estado_smart}")
    print(f"   Plazo       : {info_plazo}")
    print(f"   Avance      : {obj['avance']}%")

    if es_smart(obj):
        smart_count += 1
        avance_total += obj["avance"]

# Cálculo de indicadores agregados del SGSI
total = len(objetivos_sgsi)
porcentaje_smart = (smart_count / total) * 100 if total else 0
avance_promedio = (avance_total / smart_count) if smart_count else 0

print("\n" + "=" * 60)
print("INDICADORES AGREGADOS DEL SGSI")
print("=" * 60)
print(f"Objetivos totales         : {total}")
print(f"Objetivos SMART           : {smart_count}")
print(f"% Objetivos bien formulados: {porcentaje_smart:.1f}%")
print(f"Avance promedio (SMART)   : {avance_promedio:.1f}%")
```
